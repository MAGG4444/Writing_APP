# Story Forge

Local browser app for authors who want one workspace for drafting, outlining, plot tracking, tags, character analysis, and relationship mapping.

## Features

- Draft editor with lightweight formatting buttons and live preview
- Idea capture cards
- High-level outline and detailed scene outline pages
- Plot pointer tracker
- Tag manager
- Mind-map style relationship board for characters and plot threads
- Character panel with a dedicated analysis page per character
- Local persistence with browser `localStorage`
- JSON import/export for backups

## Run locally

Because the app is plain HTML, CSS, and JavaScript, you can either:

1. Open `index.html` directly in a browser.
2. Or serve the folder locally with a simple static server, for example:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
